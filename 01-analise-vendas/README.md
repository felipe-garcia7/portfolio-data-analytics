# Projeto: Análise de Vendas

Este projeto tem como objetivo analisar dados de vendas para identificar padrões e gerar insights de negócio.

## Objetivos da análise

- Identificar produtos mais vendidos
- Analisar faturamento por período
- Comparar vendas por região
- Identificar tendências de vendas

## Ferramentas utilizadas

- Excel
- Power BI
- SQL (futuramente)

## Status do projeto

Em desenvolvimento.
